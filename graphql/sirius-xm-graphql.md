# Sirius XM GraphQL API

The Pandora Developer API provides GraphQL-based access to Pandora's music catalog of over 30 million tracks, stations, podcasts, and playlists. It supports personalized playback, search, user feedback (thumbs up/down), collection management, and listener profile data. Authentication uses OAuth2. The API is available through the Pandora Developer Center for approved partner applications.

**Endpoint:** https://www.pandora.com/api

**Documentation:** https://developer.pandora.com/
